web design basic components

  1. html= the content of website (text & pictures)
  2. css = the style of website (colors & fonts)
  3. javascript = more advanced...the actions of website (non static information...interactive maps & animaton)

web design commands & what they do

  • HTML
    • h# - heading (h1 for largest to h6 to smallest)
    • p - paragraph of text
    • b - makes text between tags bold
    • i - makes text between tags italic
    • a href= "url"> link name /a> - creates a link
    • li - entry in list
    • ul - unordered (bulleted) list
    • ol - ordered (numbered) list
    • img src="url/filename.jpg/png" alt="alt text"> - show an image

  • CSS
    • background-color
    • color (color of text)
    • text-align
    • list-style
    • font
    • :hover

Important Things To Remember:

  1. ALWAYS close your tags "<> /> {;}"
  2. css --> selector(s) {property: value; property: value; }
  3. html --> " <> content /> "
  4. try to put style in CSS as much as possible so you dont crowd your HTML
